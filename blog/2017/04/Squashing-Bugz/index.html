Last Week in ManageIQ: Squashing Bugz

Apr 24, 2017 by h-kataria
  • LWIMIQ

Hello there! This is Harpreet Kataria, my first contribution at LWIMIQ blog post, I want to share some updates on what has been happening in ManageIQ world this past week. Team ManageIQ has been busy squashing bugz rapidly to make Fine release finer and more stable. After all bugz are bugz and must be squashed.

Count/List of PRs merged in all repos in LWIMIQ:

  • 56 - ManageIQ/manageiq
  • 59 - ManageIQ/manageiq-ui-classic
  • 29 - ManageIQ/manageiq-ui-service
  • 10 - ManageIQ/manageiq-providers-amazon
  • 2 - ManageIQ/manageiq-providers-azure
  • 6 - ManageIQ/manageiq-providers-vmware

Keep up the good work!

Improvements

Each week we strive to make ManageIQ better and better. Last week Šimon made some changes to make improvements in the seeding methods.

Šimon Do not run certain seeds twice (#14725) & (#14726)

Šimon Speed up certain seeding methods (#14721) & (#14712)

Squashing Bugz

With the amount of new features introduced in last few sprints into ManageIQ, it is possible that we introduced new bugz as well as existing bugz were discovered.

Several fixes were made in Embedded Ansible area and Ansible Services area, see list of PRs in manageiq-ui-classic & manageiq repos for full details on these.

MartinH made performance improvements with Notifications (#611)

Brian Fix Group/Role name links on User Details page (#655)

Aparna Quadicon links fixed on Tagging edit screen (#1041)

Zahi Add Container entities to TAG_CLASSES (#14535)

There were many more bugz that were fixed last week, please see complete list of merged PRs in each repo for more details.

Features

While all our developers are heads down with squashing bugz some of us managed to squeeze in some minor Enhancements this past week.

Jillian has added API support for Refresh Configuration Script Sources action (#14714).

James Added support to pickup Tower Project’s refresh status (#14687).

Milan added UI support for Embedded Ansible refresh (#934)

MartinP Custom buttons for list views (#796)

Joshua SUI - Show all custom actions vs collapsing to single dropdown (#669)

Allen SUI - Power buttons vm details (#635)

Cleanup

Aside from some new additions & lots of bug fixing, we also deleted some unwanted code from our repos.

MartinP Remove duplicate flash_msg_div (#1015) & (#1016)

Brandon Removal of unsused Global Variables from manageiq-ui-classic repo (#969)

Roman Removed unused & invalid provider documentation link (#1002)

Tzu-Mainn Remove code that creates phantom object when creating Cloud Volume & Tenant (#14766)

Jozef Removing unused host trait (#14598)

Nimrod Removed duplicate report (#14515)

Wrapping up

That’s all I have to share with you folks. Stay tuned for next week’s LWIMIQ post, by Drew.
