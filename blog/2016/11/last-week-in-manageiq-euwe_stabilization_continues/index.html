Last week in ManageIQ – Euwe stabilization continues

Nov 21, 2016 by Laura
  • LWIMIQ

Hi! This is Laura with some highlights of our Last Week in ManageIQ.

Featured

The stabilization of the upcoming Euwe release was the main focus for us last week. There were 108 PRs merged this week, containing 226 commits by 52 contributors.

If you would like to contribute, you can choose from our list of open issues.

Sprint 49 ended on Monday, November 14th, you can read all about it here. The sprint review meeting has been recorded, as always, and can be found here.

Improved

The replication exclude settings were moved out of the global settings and instead read and write directly to the pglogical configuration in the database, decoupling them from the server record.

There is now an option to hide resources in the REST API - one can use this to improve performance when only the count or subcount is needed from the query. Tim has done this work here

A middleware server running in domain mode now has different set operations than the normal standalone server available in the toolbar - PR by Jirka Kremser

Fixed

There were 69 bugs fixed this week, here is the entire list.

The repository directory for Git based domains in Automate is now removed when a domain is being deleted. Madhu made this change here

Libor fixed the calculation of average for used metrics in this PR, by using the number of hours in the period instead of count of metric rollup records.

Drew reworked the way parameters are passed into a Schedule Automate Job. See what was changed here.

New

The Gnocchi service for Openstack metrics was added by Marek in this PR, in order to support metrics in OSP10.

Dynamic dropdown list support was added by Gregor for the orchestration service dialog in this PR. This allows the VMware vCloud provider to create dropdowns for the available networks for each VM that is part of the vApp template.

Deleted

Simon and Martin were busy refactoring and deleting code - as can be seen in some of their PRs here and here, where Martin deleted a good number of lines by using an existing mixin.

Wrapping up

This was just a synopsis of last week in ManageIQ, check out our full list of merged PRs for a more complete picture.

Thanks for joining us for another edition of LWIMIQ. Have a great week!
